# 用法
        下载脚本，打开haust_drcom.py,填入帐号，密码，保存退出
        $ python haust_drcom.py 然后就可以愉快的上网了
        
 # 使用条件
        目前仅在乾5乾6测试过，可直接使用，其他的地方可能需要更新脚本中CONFIG到CONFIG_END的一些配置，参见抓包参考，有待后续测试，欢迎交流
        测试系统：kali linux / ubuntu，关于系统详情，请自行谷歌百度，或私聊交流
